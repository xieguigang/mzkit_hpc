open_mnlink {mzkit_hpc} R Documentation

Open a connection to the MySQL database for the spectrum data pool system

Description

Establishes a connection to the MySQL database used for storing spectral data and molecular networking information. Requires the graphQL package and MySQL driver.

Usage

open_mnlink(user, passwd,
    host = '127.0.0.1',
    port = 3306);

Arguments

user

Character string specifying the MySQL username for authentication.

passwd

Character string containing the password for the MySQL user.

host

Character string specifying the host address of the MySQL server. Default is "127.0.0.1" (localhost).

port

Integer specifying the port number for the MySQL connection. Default is 3306 (standard MySQL port).

Details

This function loads required packages (graphQL and hpc) and initializes a connection using the specified credentials. Ensure MySQL service is running at the target host/port.

Authors

xieguigang

Value

Returns a MySQL connection object (CLR object) that can be used for subsequent database operations. The connection is configured to use the "sample_pool" database.
